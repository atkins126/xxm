xxm Demo Projects - listprojects

This is a demo project listing the projects from a xxm.json file.

Web.xxmp
The project file. Use xxmProject.exe to edit and manage project settings.
xxmp.pas
The project's main unit. When this file is missing form the project, xxm creates one for you.
jsonDoc.pas
JSON utility unit, to read the xxm.json file. (...)
Default.xxm
The website's default page. It searches the directory listprojects.xxl is in, and the directories above it, for the xxm.json file. Loads it, and lists the "projects" elements by name (skipping any alias definitions and "xxm").

By default, a request for the root page on an xxm handler will forward to "/xxm/" (unless a "singleProject" is configured in the project registry). Register this project as "xxm" to offer a list of available projects.

Important: Use this for development servers or environments where it's safe to publish the full list of available projects on the host.
